Detailed Description

template<typename ServerOrClient>
class opcua::Node< ServerOrClient >

High-level node object to access node attribute, browse and populate address space.

The Node API is just a more convenient way of using the free functions in the services namespace.

Node objects are usefull as-is but they do not expose the entire OPC UA protocol. You can get access to the associated NodeId instance with the getNodeId() method and apply the native open62541 functions or the free functions in the services namespace.
