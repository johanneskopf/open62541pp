Detailed Description

Safe wrapper classes for heap-allocated open62541 UA_* types to prevent memory leaks.


All wrapper classes inherit from opcua::TypeWrapper. Native open62541 objects can be accessed using the opcua::TypeWrapper::handle() method.

Cast native UA_* type object references to TypeWrapper object references.

This is especially helpful to avoid copies in getter methods of composed types. A reference to a native type object be casted to a reference to a wrapper object.
